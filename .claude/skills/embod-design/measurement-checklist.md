# Measurement Checklist

Ask only for the items that materially affect the requested design.

## Printable Parts

- overall envelope or max footprint
- material and print process
- build volume constraints
- wall thickness target
- clearance / tolerance target
- mounting hole size and spacing
- required fasteners or inserts

## Mechanical Interfaces

- exact component SKU or standard
- bolt pattern
- shaft diameter / depth / flat
- bearing OD / ID / width
- cable exit and service access

## Robot Embodiment

- wheel diameter and width
- motor / gearbox model
- battery size and mass
- chassis envelope
- ground clearance
- sensor mounting positions
- links, joints, and export target (`urdf` or manufacturing only)

## Validation Intent

- expected exports
- whether a visual snapshot is needed
- whether the user wants a plan before implementation
