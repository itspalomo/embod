# Example Queries

Use these as patterns, not fixed templates.

## Clarification-First Design

- Design a PETG bracket for a NEMA17 motor. Ask for any missing measurements
  before writing code.
- Plan a wheel mount for an 8 mm shaft and a 608 bearing. Confirm tolerances,
  wheel width, and fasteners first.

## Robot Planning

- Plan a small diff-drive robot in Embod. Ask me about wheel diameter, motor
  model, battery envelope, and export requirements before implementing.
- I want a camera mast for a rover. Start in planning mode and confirm the
  sensor model, mounting interface, and allowable height.

## Validate And Iterate

- Validate this Embod project, explain any printability warnings, and show me a
  CAD snapshot.
- Refine this `embod_project.py` for 3D printing and tell me which dimensions
  you had to assume.
